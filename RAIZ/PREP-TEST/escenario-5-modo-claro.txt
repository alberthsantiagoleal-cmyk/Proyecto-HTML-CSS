================================================================
ESCENARIO 5 — MODO CLARO / OSCURO
================================================================

QUÉ ES:
  Un toggle en la pantalla de Home que cambia el tema de oscuro
  a claro. Se logra con un checkbox oculto que al marcarse
  sobreescribe las variables CSS del :root.

ARCHIVOS QUE TOCAS:
  - Editar: SCREENS/02-home.html  (agregar el toggle en el header)
  - Editar: CSS/app.css           (agregar variables del tema claro)

----------------------------------------------------------------
PASO 1 — En 02-home.html, agregar el toggle en el header
----------------------------------------------------------------

BUSCA esto:
  <section id="home">

REEMPLAZA por:
  <input type="checkbox" id="theme-toggle" class="theme-check">
  <section id="home">

BUSCA esto:
  <div class="home-header">
    <div class="avatar">👨🚀</div>
    <span>William Teig</span>
  </div>

REEMPLAZA por:
  <div class="home-header">
    <div class="avatar">👨🚀</div>
    <span>William Teig</span>
    <label for="theme-toggle" class="theme-label">☀️</label>
  </div>

IMPORTANTE: el input debe estar ANTES del section en el HTML
para que el selector ~ funcione correctamente.

----------------------------------------------------------------
PASO 2 — En app.css agregar al final
----------------------------------------------------------------

.theme-check { display: none; }

.theme-label {
  margin-left: auto;
  font-size: 1.3rem;
  cursor: pointer;
  transition: transform var(--ease);
}

.theme-label:hover { transform: rotate(20deg); }

.theme-check:checked ~ #home {
  --bg: #f0f2f5;
  --bg-card: #ffffff;
  --bg-surface: #e4e6ea;
  --text: #0d1117;
  --muted: #555e6b;
  --border: rgba(0,0,0,0.08);
  background: var(--bg);
  color: var(--text);
}

================================================================
TÉCNICA CSS USADA:
  input[type=checkbox] + :checked + selector ~
  Al marcar el checkbox, CSS sobreescribe las variables dentro
  del section #home. Como todo usa var(), el cambio se propaga
  automáticamente a todos los elementos hijos.

LIMITACIÓN:
  Solo afecta la pantalla de Home porque el checkbox vive en ese
  HTML. Para que afecte todo el proyecto necesitaría estar en
  cada página (o en un layout compartido, que no existe en HTML puro).
================================================================

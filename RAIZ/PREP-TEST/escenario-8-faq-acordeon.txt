================================================================
ESCENARIO 8 — ACORDEÓN DE PREGUNTAS FRECUENTES (FAQ)
================================================================

QUÉ ES:
  Una pantalla con preguntas y respuestas que se expanden
  al hacer clic. Cada pregunta es un label, cada respuesta
  se muestra/oculta con un checkbox + :checked.

ARCHIVOS QUE TOCAS:
  - Crear:  SCREENS/07-faq.html
  - Editar: SCREENS/02-home.html  (enlazar desde el banner)
  - Editar: CSS/app.css           (estilos del acordeón)

PASO 3 — En app.css agregar al final
----------------------------------------------------------------

#faq {
  display: flex;
  flex-direction: column;
  min-height: 100vh;
  background: var(--bg);
}

.faq-list {
  display: flex;
  flex-direction: column;
  padding: 0 20px;
}

.faq-check { display: none; }

.faq-question {
  display: flex;
  align-items: center;
  justify-content: space-between;
  padding: 16px 0;
  border-bottom: 1px solid var(--border);
  font-size: 0.95rem;
  font-weight: 500;
  cursor: pointer;
  transition: color var(--ease);
}

.faq-question:hover { color: var(--accent); }

.faq-arrow {
  font-size: 1.2rem;
  color: var(--muted);
  transition: transform 0.3s ease;
  display: inline-block;
}

.faq-answer {
  font-size: 0.85rem;
  color: var(--muted);
  line-height: 1.6;
  max-height: 0;
  overflow: hidden;
  transition: max-height 0.35s ease, padding 0.35s ease;
}

.faq-check:checked + .faq-question {
  color: var(--accent);
}

.faq-check:checked + .faq-question .faq-arrow {
  transform: rotate(90deg);
}

.faq-check:checked + .faq-question + .faq-answer {
  max-height: 200px;
  padding: 12px 0 16px;
}

================================================================
TÉCNICA CSS USADA:
  input[type=checkbox] + :checked + selector +
  La respuesta usa max-height: 0 con overflow: hidden para
  estar "colapsada". Al marcar el checkbox, CSS cambia
  max-height a 200px con transición, creando el efecto
  de expansión suave sin JavaScript.

  La flecha rota 90° con transform: rotate() al abrirse.
